About

AI・自動化を軸にしたソフトウェア開発に取り組んでいます。 Claude Code を活用した SubAgents・Skills・Commands の設計から、 Flask による REST API 構築、ユニットテストの自動化まで幅広く対応します。

「動くだけ」ではなく、セキュリティ・可読性・保守性を意識したコードを書くことをモットーにしています。 チームの生産性を底上げするワークフロー自動化を得意としています。

Works

AI / Automation

Claude Code SubAgents 構築

コードレビュー・ドキュメント生成・テスト実行など7つの専門エージェントを設計・実装。 Analyzer→Builder→Reviewer の3段階パイプラインで品質を担保。

Backend / Python

Flask REST API 開発

入力バリデーション・環境変数管理・エラーハンドリングを組み込んだ セキュリティ強化済みの Python Web API。SQLite と連携した CRUD エンドポイントを提供。

Testing / CI

自動テストパイプライン

Python unittest を用いた23ケースのユニットテスト自動化。 エッジケース・境界値・型エラーまで網羅し、リグレッションを防止する堅牢なテスト基盤を構築。

AI / Content

SEO ブログ自動生成

Claude Code の seo-blog-writer エージェントを活用し、 キーワード選定から記事執筆・構成最適化まで自動化したコンテンツ生成ワークフロー。
